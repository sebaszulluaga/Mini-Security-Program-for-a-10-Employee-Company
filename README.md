# Mini Security Program for a 10-Employee Company

This project simulates the implementation of a basic cybersecurity program for a small company using Microsoft 365 and Microsoft Security concepts.

## Objectives

- Implement basic Identity and Access Management controls.
- Define MFA and Conditional Access policies.
- Create a risk matrix.
- Design a basic incident response plan.
- Propose monitoring activities for a small SOC/security function.
- Align security controls with business needs.

## Skills Demonstrated

- Identity and Access Management
- Microsoft Entra ID concepts
- MFA and Conditional Access
- Risk management
- Incident response
- Security documentation
- Microsoft Security fundamentals

## Target Roles

- IAM Analyst Junior
- Security Analyst Junior
- SOC Analyst Junior
- IT Security Analyst
- Cloud Security Analyst Junior
